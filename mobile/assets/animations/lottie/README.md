# Lottie Animation Files

## Required Lottie Files

Download the following Lottie animation files from [LottieFiles.com](https://lottiefiles.com/) and place them in this directory:

### 1. loading.json
**Purpose:** Loading spinner animation
**Usage:** Replace CircularProgressIndicator throughout the app
**Recommended:** Search for "loading spinner" or "loading dots"
**Size:** 100x100px
**Loop:** Yes
**Suggested:** https://lottiefiles.com/animations/loading

### 2. empty_state.json
**Purpose:** Empty list states (bookings, guides, etc.)
**Usage:** Show when lists have no data
**Recommended:** Search for "empty box" or "no data"
**Size:** 200x200px
**Loop:** No
**Suggested:** https://lottiefiles.com/animations/empty-state

### 3. success.json
**Purpose:** Booking confirmation success
**Usage:** Display after successful booking
**Recommended:** Search for "success checkmark" or "celebration"
**Size:** 150x150px
**Loop:** No
**Suggested:** https://lottiefiles.com/animations/success

### 4. error.json

### 5. search_animation.json

## How to Download

1. Visit [LottieFiles.com](https://lottiefiles.com/)
2. Search for the animation type (e.g., "loading spinner")
3. Choose a free animation that matches the GIDYO brand (teal/navy colors preferred)
4. Click "Download" and select "Lottie JSON"
5. Rename the file to match the names above
6. Place in this directory

## Color Customization

If the animation colors don't match the GIDYO brand, you can:
1. Use the LottieFiles editor to change colors before downloading
2. Choose animations that use neutral colors (black/white/gray)
3. Look for animations specifically in teal (#0D9488) or navy (#1E3A5F)
